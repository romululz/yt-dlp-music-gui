## yt-dlp-music-gui

GUI frontend for [link to yt-dlp-music] using Zenity

### Dependencies

```
sudo pacman -Sy yt-dlp ffmpeg python-mutagen zenity git base-devel
```

### Manual Installation

Open terminal
```
git clone https://github.com/YOURNAME/yt-dlp-music-gui.git
```

```
cd yt-dlp-music-gui
```

```
sudo make install
```


### Uninstall

```
sudo make uninstall
```
